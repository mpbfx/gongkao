# 公考题库 UI/UX 重设计：墨线校准

## 设计概念

“墨线校准”把产品定义为一套专注、精密、可信赖的公考训练仪器。视觉灵感来自高质量编辑出版物、考试答题纸和红笔校对，但不复刻传统书卷装饰。界面通过清晰的信息密度、稳定的网格、克制的状态色和可追溯的学习反馈，让用户始终知道“现在做什么、为什么做、下一步是什么”。

## 视觉系统

- 主背景：暖纸白 `#F4F0E8`
- 主要文字与导航：深墨蓝 `#172238`
- 核心操作与强调：朱砂橙 `#D65A3A`
- 正确、掌握、完成：校对绿 `#2E7665`
- 提醒、待复盘：赭金 `#B8842F`
- 边框与分隔：灰蓝墨线 `#C8CED2`
- 标题字体：思源宋体 / Source Han Serif SC，强调稳重与编辑感
- 正文字体：思源黑体 / Noto Sans SC，保证小字号和数据可读性
- 数字字体：等宽数字，使用表格数字特性
- 圆角：控件 8px、信息卡 14px、重点容器 20px；后台密集表格减少圆角
- 图标：统一 1.75px 线性图标；状态不可只依赖颜色，配合图标、文案和形状

## 响应式原则

- 桌面端：学生端使用可收起的左侧导航；做题和错题复盘使用双栏或三栏工作台；后台使用固定管理侧栏与高密度内容区。
- 移动端：底部五项导航；主要操作固定在拇指可达区域；筛选、答题卡、助教和详情进入底部抽屉；避免横向表格。
- 所有正文保持至少 4.5:1 对比度；触控目标不小于 44px；焦点态使用高对比墨蓝外圈与朱砂定位点。

## 当前实现参考范围

1. 首页（访客与已登录状态）
2. 登录页（默认、错误与提交中）
3. 我的工作台
4. 历年试卷列表（筛选、空、加载、错误）
5. 试卷详情
6. 专项组卷（未选择、已选择、校验错误）
7. 做题工作台（作答、暂停、答题卡、离线）
8. 练习结果与历史回看（解析、助教、草稿回看）
9. 复盘记录（筛选、空、加载）
10. 错题复盘工作台（队列、题目上下文、助教、移动端详情）
11. 错因报告（趋势、分布、矩阵、空数据）
仅保留 01–11 作为现有学生端桌面实现的视觉参考。管理端与移动端不属于本轮改造范围；实现时以桌面画面为依据，并继续保留项目既有的移动端代码和交互。

设计稿中的蓝色或紫色偏差不作为实现依据。跨页面统一采用暖纸白、深墨蓝、朱砂橙、校对绿和赭金组成的“墨线校准”视觉系统。

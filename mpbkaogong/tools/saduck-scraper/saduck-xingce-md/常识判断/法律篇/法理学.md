# 法理学

## 一、法的概念

### 法的含义与特征

1.  **1、法的含义**：法是由国家制定、认可并依靠国家强制力保证实施的，以权利和义务为调整内容，以人的行为及行为关系为调整对象，反映由特定物质生活条件所决定的统治阶级或人民意志的行为规范体系。

    1.  （1）产生方式：法是随着国家、随着阶级的产生而出现的。`先有国家再有法律`
    2.  （2）保障力量：国家强制力是保障法实施的最后保障。
    3.  （3）决定因素：特定物质生活条件。`经济基础决定上层建筑，法属于上层建筑`
    4.  （4）法的本质：体现统治阶级意志。`我国的统治阶级是人民`
    5.  （5）法的内容：以权利和义务为基础。
    6.  （6）行为规范：调整行为。`不调整思想`
2.  **2、法的特征**：

    1.  （1）国家意志性。法是由国家制定和认可的。法是被奉为法律的统治阶级意志，意味着法是国家制定和认可的，是国家意志的体现，具有普遍的约束力。
    2.  （2）权利义务性。权利义务构成了法律的主要内容，为人们的行为设定了“权利模式”和“义务模式”。
    3.  （3）国家强制性。法是由国家强制力保证实施的。国家的强制力不同于其他组织的强制力。国家强制力掌握在统治阶级手中，针对的主要是被统治阶级力。国家强制性是法区别于道德、宗教教规等其他社会规范最主要的特征。
    4.  （4）普遍性。法在其管辖范围内对不特定的多数人普遍适用，“法律面前人人平等”、“王子犯法与庶民同罪”即体现了法的普遍性。
    5.  （5）规范性。法是调整人的行为的社会规范，通过为人的行为设定行为模式，规定了人们可以做什么，应当做什么，不得做什么，来间接调整社会关系。
    6.  （6）程序性。法是保障人们权利与义务的规范体系，则其本身必然有一定的程序要求，例如各类纠纷所引起的诉讼有非常多的程序要求。
    7.  （7）可诉性。法是一种可以被诉诸法律程序的行为规范。当个人或组织认为自己的权利受到侵害时，可以通过法律程序寻求救济，这是法具有可诉性的直接体现‌。

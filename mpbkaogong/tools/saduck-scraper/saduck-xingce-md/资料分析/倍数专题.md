# 倍数专题

1.  **1、增长率和倍数的区别**：

    1.  （1）增长率是比基期多出的比率（识别：A比B增长(多) .... % 或者 几倍），而倍数是指两数的直接比值（识别：A是B的 .... 倍），不能单纯的通过 % 或 倍 去区分，而是看“谁比谁增长”或“谁是谁的...倍”

    2.  （2）增长率计算公式为（现期值/基期值）-1。倍数计算公式为 现期值/基期值。

    3.  （3）当现期值是基期值的N倍时，其增长率为N-1（例如2倍对应的增长率为100%）。公式表达：‌增长率=倍数-1 或 ‌倍数=增长率+1。‌

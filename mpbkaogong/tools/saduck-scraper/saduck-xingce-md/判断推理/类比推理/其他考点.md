# 其他考点

## 一、人与物

**例**：（2024四川）编剧：导演：电影

1.  A.推广：运营：广告
2.  B.内容：校对：文章
3.  C.经理：编程：产品
4.  D.编辑：设计：图书

解析

1.  编剧和导演均为制作电影的不同职业，且编剧对电影的内容负责，导演对电影的呈现负责，二者分别与电影构成对应关系。
2.  A项：广告是推广的一种方式，推广不是制作广告的身份，与题干逻辑关系不一致，排除；
3.  B项：内容是文章的组成部分，不是制作文章的身份，与题干逻辑关系不一致，排除；
4.  C项：产品和编程没有必然联系，编程不一定是制作产品的身份，与题干逻辑关系不一致，排除；
5.  D项：编辑和设计均为制作图书的不同身份，且编辑对图书的内容负责，设计对图书的呈现负责，二者分别与图书构成对应关系，与题干逻辑关系一致，当选。
6.  故正确答案为D。

**例**：（2024江苏）球赛：球员：裁判

1.  A.教学：教师：家长
2.  B.庭审：被告：法官
3.  C.球场：球员：观众
4.  D.手术：患者：护士

解析

1.  “球员”参与“球赛”，二者为主体与活动的对应关系；“裁判”判罚“球员”，二者为主体与被判罚对象的对应关系。
2.  A项：“教师”参与“教学”，二者为活动与主体的对应关系，但“家长”不能判罚“教师”，与题干逻辑关系不一致，排除。
3.  B项：“被告”参与“庭审”，二者为主体与活动的对应关系；“法官”判罚“被告”，二者为主体与被判罚对象的对应关系。与题干逻辑关系一致，当选。
4.  C项：“球员”在“球场”打球，二者是地点与主体的对应关系，与题干逻辑关系不一致，排除。
5.  D项：“患者”接受“手术”，而不是参与“手术”，且“护士”不能判罚“患者”，与题干逻辑关系不一致，排除。
6.  故正确答案为B。

**例**：（2024山东）颁奖人：获奖者

1.  A.参会人：主持人
2.  B.纳税人：税务局
3.  C.纪录片：观众
4.  D.绑架者：人质

解析

1.  颁奖人颁奖的对象是获奖者，二者是动作发出者与对象的对应关系。
2.  A——参会人和主持人不是动作发出者与对象的对应关系，排除；
3.  B——纳税人纳税的对象是税务局，二者是动作发出者与对象的对应关系，保留；
4.  C——观众观看的对象是纪录片，二者是动作发出者与对象的对应关系，但词语顺序与题干相反，且纪录片非人，排除；
5.  D项——绑架者绑架的对象是人质，二者是动作发出者与对象的对应关系，与题干逻辑关系一致，保留。
6.  对比B、D两项，题干中的“获奖者”和D项中的“人质”均指“人”，而B项中的税务局是“政府机关”，故选D。

**例**：（2019天津某中心）消防员：灭火：灭火器

1.  A.护士：手术：手术刀
2.  B.厨师：炒菜：炒勺
3.  C.老师：板书：粉笔
4.  D.农民：种地：犁

解析

1.  消防员是一种职业，灭火是消防员的主要工作，灭火器是灭火的工具，三者为职业、工作内容、工具（原指工作时所需用的器具）的对应。
2.  A项：用手术刀做手术的职业应为医生，而非护士，与题干逻辑关系不一致，排除；
3.  B项：厨师是职业，炒菜是主要工作，炒勺是炒菜的工具，三者为职业、工作内容、工具，与题干逻辑关系一致，当选；
4.  C项：老师是职业，板书是工作内容，粉笔写板书的工具。但板书不是老师的主要工作，老师的主要工作是教书，与题干逻辑关系不一致，排除；
5.  D项：农民是职业，种地是农民的主要工作，但犁是用来耕地的，而不是用来种地的，与题干逻辑关系不一致，排除。
6.  故正确答案为B。

**例**：（2016河南三门峡事业单位）“杂志” 对于 （ ） 相当于 （ ） 对于 “农民”

1.  A.书刊；农村
2.  B.编辑；蔬菜
3.  C.传媒；农业
4.  D.报纸；果农

解析

1.  分别将选项代入题干，确定逻辑关系。
2.  A项：“杂志”是“书刊”的一种，二者属于包容关系中的种属关系；“农民”生活在“农村”，二者属于地点对应关系，前后逻辑关系不一致，排除。
3.  B项：“编辑”编写“杂志”，二者属于主宾关系；“农民”种植“蔬菜”，二者属于主宾关系，前后逻辑关系一致，符合。
4.  C项：“杂志”是一种“传媒”，二者属于包容关系中的种属关系；“农业”与“农民”属于对应关系，前后逻辑关系不一致，排除。
5.  D项：“杂志”与“报纸”属于并列关系；“果农”是“农民”的一种，二者属于包容关系中的种属关系，前后逻辑关系不一致，排除。
6.  因此，选择B选项。

**例**：（2019江苏常州事业单位）七夕∶织女

1.  A.除夕∶晚会
2.  B.清明∶先烈
3.  C.重阳∶茱萸
4.  D.端午∶屈原

解析

1.  “七夕”是中国传统节日，对应的人物是“织女”，并且“织女”为个体，二者属于节日与人物的对应关系。
2.  A项：“除夕”是中国传统节日，会举办“晚会”，但“晚会”不是人物，排除；
3.  B项：“清明”是中国传统节日，会祭奠“先烈”，但“先烈”是群体概念，排除；
4.  C项：“重阳”是中国传统节日，对应的习俗是插“茱萸”，但“茱萸”不是人物，排除；
5.  D项：“端午”是中国传统节日，对应的人物是“屈原”，并且“屈原”为个体，二者属于节日与人物的对应关系，与题干逻辑关系一致，符合。
6.  因此，选择D选项。

## 二、公式

**例**：（2023国考）小计：总计

1.  A.平均值：总值  B.被乘数：总数
2.  C.单科分：总分  D.分目录：总目

解析

2.  前者相加等于后者，C 最符合，所有“单科分”相加可以得到“总分”。易错项是 D, “分目录”指在各部分之前的目录，“总目”指在整个文档之前的目录，“总目”是对“分目录”的汇总，不存在相加的计算关系。

**例**：（2018浙江选调）出口：进口：贸易顺差

1.  A.征税：纳税：财政收入
2.  B.买进：卖出：盈利
3.  C.支出：收入：赤字
4.  D.还债：借债：债务

解析

3.  当（出口-进口）>0时为贸易顺差
4.  A项：征税是国家征收税务，纳税是纳税人缴纳税务，只要国家征税了，纳税人纳税了就会产生财政收入，跟二者大小没有关系，与题干逻辑关系不一致，排除；
5.  B项：（卖出-买进）>0时为盈利，买进与卖出的顺序与题干顺序相反，与题干逻辑关系不一致，排除；
6.  （支出-收入）>0时为赤字，与题干逻辑关系一致，当选；
7.  D项：只要借债了就会产生债务关系，借债大于还债会说明还有债没还清，依然存在债务关系，但是还债与借债的顺序与题干顺序相反，与题干逻辑关系不一致，排除。
8.  故正确答案为C。

**例**：（2014浙江）路程：时间：速度

1.  A.压强：面积：压力
2.  B.功率：电流：电压
3.  C.本金：利率：利息
4.  D.原价：折扣：现价

解析

3.  路程等于时间乘以速度。
4.  压强等于压力除以面积，A项错误；
5.  功率等于电流乘以电压，B项正确；
6.  利息等于本金乘以利率，C项词语间位置错误；
7.  现价等于原价减去折扣，D项错误。
8.  故正确答案为B。

## 三、字义

**例**：（2025天津）备：德才兼备：戒备森严

1.  A.狼：狼吞虎咽：狼奔豕突
2.  B.兵：短兵相接：草木皆兵
3.  C.绳：结绳而治：绳锯木断
4.  D.假：狐假虎威：不假思索

解析

1.  “德才兼备”指既有好的思想品质，又有工作的才干和能力，其中“备”的意思是“具备”；“戒备森严”指警戒防备极其严密，其中“备”的意思是“防备”。“备”在后两词中的含义不同。
2.  A项：“狼吞虎咽”指像狼和虎那样吞咽东西；“狼奔豕突”指如狼和猪那样奔跑冲撞，后两词中“狼”的意思均是“像狼那样”，含义相同，与题干逻辑关系不一致，排除；
3.  B项：“短兵相接”指敌我逼近，用短兵器交战，其中“兵”的意思是“兵器”；“草木皆兵”指把山上的野草和树木都当作敌兵，其中“兵”的意思是“战士、军队”，“兵”在后两词中的含义不同，与题干逻辑关系一致，当选；
4.  C项：“结绳而治”指上古没有文字，用结绳记事的方法治理天下，其中“结绳”指将两根绳子扎接起来；“绳锯木断”指以绳当锯，也能锯断木头，后两词中“绳”的意思均是“用两股以上的棉麻纤维或棕草等拧成的条状物”，含义相同，与题干逻辑关系不一致，排除；
5.  D项：“狐假虎威”指狐狸借着老虎的威风去吓唬其他野兽；“不假思索”形容做事答话敏捷、熟练，用不着考虑，后两词中“假”的意思均是“借用、利用”，含义相同，与题干逻辑关系不一致，排除。
6.  故正确答案为B。

**例**：（2024江苏）海：海沟：海量

1.  A.天：天眼：天堑
2.  B.碑：碑座：碑林
3.  C.火：火焰：火气
4.  D.心：心房：心腹

解析

1.  “海沟”指深度超过6000米的狭长的海底凹地，其中“海”是指海洋，是海的本义；“海量”指极大的数量、很大的酒量或宽宏的度量，其中“海”指像海一样，是海的引申义。二者含义不同，且第二个词为本义，第三个词为引申义。
2.  A项：“天眼”指佛教所说五眼之一、道教所说天神之眼，其中“天”指天空，是天的本义；“天堑”指天然形成的隔断交通的大沟，其中“天”指天然的，不是天的引申义，与题干逻辑关系不一致，排除；
3.  B项：“碑座”指石碑下边的底座，“碑林”指聚集在一起的众多石碑，其中“碑”均指刻着文字或图画，竖起来作为纪念物或标记的石头，均是碑的本义，与题干逻辑关系不一致，排除；
4.  C项：“火焰”指燃烧着的可燃气体，其中“火”指物体燃烧时所发出的光和焰，是火的本义；“火气”多指怒气、暴躁的脾气，其中“火”指像火一样，是火的引申义。二者含义不同，且第二个词为本义，第三个词为引申义，与题干逻辑关系一致，当选；
5.  D项：“心房”指心脏内部上面左右两个空腔，也指人的内心，其中“心”是指心脏，是心的本义；“心腹”指亲信的人，其中“心”不是指像心一样，不是心的引申义，与题干逻辑关系不一致，排除。
6.  故正确答案为C。

## 四、常识

**例**：（2017湖北）虚数：复数

1.  A.法律：刑法
2.  B.麦子：水稻
3.  C.祥林嫂：《祝福》
4.  D.人工智能：计算机科学

解析

1.  复数是指由实数部分和虚数部分所组成的数。当虚数部分等于零时，这个复数就是实数；当虚数部分不等于零时，这个复数称为虚数。
2.  也就是说所有的虚数都是复数，二者是包容关系中的种属关系。
3.  A项：刑法是法律的一种，二者是包容关系中的种属关系，但是词的顺序反了，与题干逻辑关系不一致，排除；
4.  B项：麦子和水稻是并列关系，与题干逻辑关系不一致，排除；
5.  C项：祥林嫂是小说《祝福》中的人物，是逻辑关系中的对应关系，与题干逻辑关系不一致，排除；
6.  D项：人工智能是计算机科学的一种，二者是包容关系中的种属关系，与题干逻辑关系一致，当选。
7.  故正确答案为D。
8.  `复数包括实数和虚数；实数包括有理数和无理数；有理数包括整数和分数；无理数包括正无理数、负无理数；整数包括正整数、0、负整数；分数包括正分数、负分数。`

**例**：（2017河南）无机物：化合物

1.  A.中草药：野菜
2.  B.二氧化碳：有机盐
3.  C.空间：时间
4.  D.原子：电子

解析

1.  无机物是单质和无机化合物的统称。化合物分为无机化合物和有机化合物。二者之间是交叉关系。
2.  A项：野菜是可以做蔬菜的野生植物，中草药即为中药，是中医所用的药物，以植物为最多，也包括动物和矿物。二者之间是交叉关系，与题干逻辑关系一致，当选；
3.  B项：有机盐是有机酸、生物碱适当中和生成的盐，二氧化碳和有机盐没有必然的逻辑关系，与题干逻辑关系不一致，排除；
4.  C项：空间是与时间相对的一种物质客观存在形式，二者之间是对应关系，与题干逻辑关系不一致，排除；
5.  D项：原子由原子核和绕核运动的电子组成，电子是原子的组成部分，二者为组成关系，与题干逻辑关系不一致，排除。
6.  故正确答案为A。

**例**：（2017吉林）植物：裸子植物：红杉

1.  A.学校：中学：小学
2.  B.国家：民族：公司
3.  C.科学：物理学：力学
4.  D.星系：星座：太阳

解析

1.  裸子植物是一种植物，红杉是一种裸子植物，都是种属关系。
2.  A项：中学是一种学校，种属关系，小学与中学并列，与题干逻辑关系不一致，排除；
3.  B项：国家由多个民族组成，组成关系，公司与民族无必然联系，与题干逻辑关系不一致，排除；
4.  C项：物理学是一种科学，力学是属于物理学，都是种属关系，与题干逻辑关系一致，当选；
5.  D项：星系是由几千亿可恒星组成的巨大恒星系统，星座是表示星空中方位的区域，两者无必然联系，太阳是星系中的一部分，组成关系，与题干逻辑关系不一致，排除。
6.  故正确答案为C。
7.  `植物按照有种子和没种子，分为种子植物和孢子植物。孢子植物没有种子，靠孢子繁殖，如：藻类、苔藓类、蕨类。`
8.  `种子植物根据种子外有无果皮包被，分为被子植物（如：桃树、杏树）和裸子植物（如：松、杉、柏、银杏、铁树）。`

**例**：（2015云南26%）三顾茅庐：刘备

1.  A.负荆请罪：蔺相如
2.  B.请君入瓮：周兴
3.  C.雪中送炭：宋太宗
4.  D.程门立雪：程颐

解析

1.  在三顾茅庐的典故中，刘备为典故的主体，诸葛亮为典故的客体。
2.  A项，蔺相如为负荆请罪的客体，主体为廉颇，排除；
3.  B项，周兴为请君入瓮的客体，主体为来俊臣，排除；
4.  C项，雪中送炭的主体为宋太宗，与题干逻辑关系相符；
5.  D项，程颐为程门立雪的客体，主体为杨时，排除。
6.  故正确答案为C。
7.  | 典故 | 主体 | 客体 |

    | 负荆请罪 | 廉颇 | 蔺相如 |
    | 请君入瓮 | 来俊臣 | 周兴 |

3.  蚜虫会吸食小麦的汁液，二者为捕食者与被捕食对象的对应关系；七星瓢虫会捕食蚜虫，二者为捕食者与被捕食对象的对应关系。
4.  A项：有的蛇会吃鸟，有的鸟会吃蛇，二者是相互捕食的对应关系，与题干逻辑关系不一致，排除；
5.  B项：稗草是一种常见的田间杂草，会和水稻竞争稻田里的资源，二者是竞争关系，并非捕食者与被捕食对象的对应关系，与题干逻辑关系不一致，排除；
6.  C项：蚊子和高粱没有必然关系，与题干逻辑关系不一致，排除；
7.  D项：老鼠吃玉米，二者为捕食者与被捕食对象的对应关系；黄鼬即黄鼠狼，会捕食老鼠，二者为捕食者与被捕食对象的对应关系，与题干逻辑关系一致，当选。
8.  故正确答案为D。

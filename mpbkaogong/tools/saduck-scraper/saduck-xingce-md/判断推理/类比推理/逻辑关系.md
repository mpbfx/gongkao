# 逻辑关系

## 一、全同关系

1.  **1、定义**：A与B两个概念指的是同一事物，只是称谓不同。

    1.  `例如`：
    2.  ①父亲：爸爸
    3.  ②西红柿：番茄
    4.  ③麦克风：话筒
    5.  ④苏州：姑苏（姑苏是苏州的别称）
    6.  ⑤刘彻：汉武帝
    7.  ⑥犬子：儿子
2.  **2、积累**：

|  |  |  |  |
| --- | --- | --- | --- |
| 扬州-广陵 | 杭州-临安 | 合肥-庐州 | 开封-汴京 |
| 重庆-山城 | 济南-泉城 | 福州-榕城 | 广州-羊城/花城 |
| 荷花-芙蕖/菡萏 | 银杏-公孙木 | 杜鹃花-映山红 | 杜鹃鸟-子规 |
| 蟋蟀-促织 | 妻子-贱内/拙荆/内人 | 元宵节-上元节 | 七夕节-乞巧节 |
| 风筝-纸鸢 | 寺院-珈蓝 | 白酒-烧酒 | 书信-尺素 |
| 分母-除数 | 百分比-百分率 | 乙醇-酒精 | 岳父-泰山 |
| 女婿-东床 | 粽子-香黍 | 等边三角形-正三角形 | 橘子皮-陈皮/红皮 |
| 人贩子-牙婆 | 生石灰-氧化钙 | 李贺-诗鬼 | 往昔-从前 |

**例**：（2022浙江）考拉：树袋熊

1.  A.墨鱼：乌贼
2.  B.蜜蜂：黄蜂
3.  C.蚱蜢：蝗虫
4.  D.金龟子：天牛

解析

5.  考拉指树袋熊，两者为全同关系。
6.  A项，墨鱼一般指乌贼，两者为全同关系，与题干逻辑关系一致。
7.  而B项，蜜蜂和黄蜂是两种不同的蜂；C项，根据农百科的解释，蚱蜢和蝗虫的类别不同；D项，金龟子和天牛是两种不同的昆虫；所以BCD中都不是全同关系，与题干逻辑关系不一致。
8.  故答案为A。

**例**：（2020海南）牵牛花：喇叭花

1.  A.乞巧节：七夕节
2.  B.七巧板：橡皮泥
3.  C.人行道：车行道
4.  D.防腐剂：添加剂

解析

5.  牵牛花又称喇叭花，二者为全同关系。
6.  A项：乞巧节又称七夕节，二者为全同关系，与题干逻辑关系一致，当选。
7.  B项：七巧板和橡皮泥均为儿童玩具，二者为并列关系，与题干逻辑关系不一致，排除；
8.  C项：人行道和车行道为并列关系，与题干逻辑关系不一致，排除；
9.  D项：防腐剂是添加剂的一种，二者为种属关系，与题干逻辑关系不一致，排除。
10.  故答案为A。

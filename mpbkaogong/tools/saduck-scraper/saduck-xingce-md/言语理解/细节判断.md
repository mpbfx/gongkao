# 细节判断

> 细节判断题的做题方法很像“大家来找茬”这个游戏，它有的题型比较简单，需要我们与原文进行一一对应，但是有的题就会难一些，不光需要对应原文，符合原文的逻辑，同时也要结合文段要对文段有准确的理解。

## 一、提问方式

1.  **1**、以下对文段理解正确/不正确的是……

2.  **2**、符合/不符合这段话意思的是……

3.  **3**、从文段中可以得知/推出的是……

4.  **4**、关于……，上文提到了/没有提及……

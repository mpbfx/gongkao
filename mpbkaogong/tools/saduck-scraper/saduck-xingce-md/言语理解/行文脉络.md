# 行文脉络

1.  中心理解题提问方式：

    1.  （1）这段文字主要/ 旨在/ 重在/ 意在/ 想要说明（ 论述，强调） 的是.....
    2.  （2）这段文字的主旨/ 主题/ 观点是.......
    3.  （3）对这段文字概括最恰当的一项是.......
2.  中心理解题基本是找文章的主旨，文章主旨指作者通过文字表达的核心观点或主要对策，是全文的核心。

3.  梳理行文脉络是言语理解最基本的做题方式，通过了解总句、分句的特征，找出文段中心句，确定正确选项。

4.  当所有技巧（转折、并列...）都不能锁定正确答案或者本身基础薄弱无法应用技巧时，梳理行文脉络就是破题方法。

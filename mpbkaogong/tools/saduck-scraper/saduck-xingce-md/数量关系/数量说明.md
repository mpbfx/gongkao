# 前言

1.  **题量**：10/15道题

2.  **题型分布**：数字推理（给出数列找规律）、数字运算（小初中奥数题）

3.  **基础目标**：正确率大于等于50%，尽量把简单的做了，难的靠蒙。

4.  **进阶目标**：正确率在75%以上

## 一、题目特点

1.  **多样性**：数量关系题型多样，包括数字推理、数学运算（几何问题、概率问题等），覆盖了数学的多个领域。

2.  **灵活性**：题目设计灵活，往往结合实际情境，要求考生在短时间内迅速理解题意并作出准确判断。

3.  **难度梯度**：题目难度从易到难，既有基础题也有难题。我们要把其中简单一些的题一定要认真做完，把那些复杂的题目一定要果断放弃。所以建议普通的考生在考场上，做一半数量关系，蒙一半数量关系。

## 二、如何备考

1.  > 为什么80%公考人觉得数量关系难？主要有两种情况：
    > （1）题干太长，给了一堆条件，读完第一遍觉得脑子里大雪无痕，处于恍惚状态，只能再读第二遍。
    > （2）题干太短，信息量太少，就好像看了一部被删减得残缺不全的电影，看完心里直呼：这就完了？

2.  数量关系每道题用到的方法大多只有一种，所以不要想得太复杂。先将数量关系基础过一遍，第一遍学习注意做基本题，要保证基本题不错。如何保证基本题不会做错呢？很简单，记住公式就可以了。

3.  基础题搞定之后，拔高题其实就是基础题的变种。学习的时候，不贪多，考频低的题型就少做一些，考频高的就多学几道变种题。 `什么是变种题？就是题干多了一些信息，多了一些计算步骤。`

4.  变种题做完之后，再攻克国考的难题。

5.  另外，着重吃透解题思想，也就是提高你对数字的敏感程度，这种敏感程度，能让你蒙题都比别人准，做题比别人快。

## 三、做题步骤

1.  > 在考场上为了节省时间，建议把数量排在第二或者第三顺位。如果你把数量放最后，可能会时间来不及，容易恐慌，导致静不下心来。

2.  第一步，读题，主要看问题和选项，先圈出必做题，也就是自己有绝对把握的题；然后圈出可能会的题，就是步骤可能有些麻烦。

3.  第二步，做题，尽量能用选项代入的用选项代入。拿不准就列方程，求谁设谁；没给具体数值就赋值；题干复杂就列表。

4.  第三步，蒙题，难度较大的题，有思路就做，没思路可根据ABCD选项平均布局来进行蒙。 `如果一个题都没写建议只蒙一个选项`

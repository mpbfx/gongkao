# 文章写作

## 一、评分标准参考

1.  > 申论大作文评分重点主要体现在立意或观点、结合资料的程度，结构层次、语言表达和卷面整洁程度等几个方面。一般国考申论大作文是35分，比照换算即可。

2.  **1、一类文（31—40分）**：

    1.  （1）立意明确、观点深刻：能围绕主题展开议论，可以论述意义、做法等，观点深刻，可置身于整个社会发展的高度，透析其本质。

    2.  （2）展开思考、多维分析：分析角度尽量多元，可以从两个以上的角度，透析意义、做法等。

    3.  （3）内容充实、论述充分：对材料的运用可靠、丰富，逻辑严密，详略得当；论述中要紧密结合时代背景，凸显顺应时代。

    4.  （4）规范严谨、符合体例：结构完整、严谨，标题规范，语言流畅，文字精炼，符合体例（议论文体例，不能写成记叙文等），书写规范、工整，格式正确，无错别字，标点正确，卷面整洁、美观。

3.  **2、二类文（21—30分）**：

    1.  （1）立意明确、观点较为鲜明：能围绕主题展开议论，可以论述其中的意义、做法等，能够站在社会发展的高度认识问题。

    2.  （2）分析维度多元。可以选择多个不同层面，对主题展开分析，看待问题全面、 多元。

    3.  （3）内容充实，论述清楚：能够从不同角度看问题，材料可靠，层次分明，有重点和说服力；论述中可结合时代背景。

    4.  （4）结构匀称、语言流畅：结构完整、匀称，语言通顺，符合体例，字迹清晰，符合规定字数；但在语言表达方面缺少亮点。

4.  **3、三类文（11—20分）**：

    1.  （1）不存在四类文中的现象。

    2.  （2）主题正确，观点不够深刻。能够围绕主题展开论述，但观点不够明确，让人难以理解；或论述角度过于微观，如仅站在单方面角度开展论述；或论述内容过于庞杂。

    3.  （3）分论点模式化倾向严重，观点空洞的。

    4.  （4）内容单薄，论述空泛。能够围绕观点展开基本论述，但不能紧扣时代背景和主题进行论述。

    5.  （5）结构基本完整，有语病，字迹潦草，接近规定字数。

5.  **4、四类文（6—10分）**：

    1.  （1）偏离题意，脱离主题。

    2.  （2）观点背离材料，对主题持否定态度的。

    3.  （3）结构不完整或者混乱，不符合文体要求的。

    4.  （4）语言不通顺，语病多的。

    5.  （5）字迹难以辨认，字数 600 字以下的。

    6.  （6）抄袭原文达四分之一以上的。

6.  **5、其他扣分说明**：

    1.  （1）没有题目，酌情扣一至两分。

    2.  （2）关键字出现错别字1个扣1分，重复不计；3个错别字扣1分。字数 50 字以内的不扣分，100 字以内的扣1分，200 字以内的内容再好最多定为三类。

    3.  （3）在行文中写无关文字。如有考生整个文章写完再最后加上一段话写给考官，这种直接打入四类。

    4.  （4）标点错误较多或书写模糊扣1分；

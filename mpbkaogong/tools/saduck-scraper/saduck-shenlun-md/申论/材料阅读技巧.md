# 材料阅读技巧

1.  **1、申论材料的本质特征**：申论材料是经过精心编排的“半成品”，看似松散，实则内在逻辑严密。它模拟了公务员日常处理文件、梳理信息的工作场景，考查的是你从杂乱信息中抓重点、析本质的能力。

2.  **2、申论材料阅读的误区**：拿到材料就从头读到尾，逐字逐句划重点，结果读完脑子一团乱麻。要么把案例细节当重点，记住某村名字却漏了核心问题。要么被数据绕晕，盯着百分比计算，却不知数字是说明“发展快”还是“差距大”。

# 加强削弱

## 一、基础

1.  **1、加强削弱的本质**：
    1.  【`例`】小明扶老奶奶过马路，所以小明是个好人。

        1.  A.小明不是好人。
        2.  B.小明确实是个好人。
        3.  C.小明还曾背着老奶奶过马路。
        4.  D.小明还曾骑着老奶奶过马路。
        5.  E.扶老奶奶过马路的都是好人
    2.  【`分析`】小明扶老奶奶过马路是论据，小明是个好人是论点，论点和论据话题不一致。下面逐一分析选项。

        1.  A：小明不是好人。直接否定论点。所以明显是个削弱项。
        2.  B：小明确实是个好人。直接肯定论点。是加强项。其实这个选项是没有意义，因为你重复了论点。
        3.  C：小明除了扶老奶奶过马路。小明还曾背着老奶奶过马路，补充了论据，能够增加小明是个好人的概率。是加强项。
        4.  D：小明还曾骑着老奶奶过马路。否定论据，能够降低小明是个好人的概率。是削弱项；
        5.  E：扶老奶奶过马路的都是好人。“扶老奶奶”和“好人”建立联系，可以理解为桥梁，属于加强项。如果你要削弱他，那就把这个桥梁切断，即“扶老奶奶过马路的都不是好人”。
    3.  【`综上`】：

        1.  加强论证：就是要增大论证成立的概率。可以证明论点或论据的真实性，也可以强化论点和论据间的联系。
        2.  削弱论证：就是要减小论证成立的概率。可以指出论点和论据的错误，也可以弱化论点和论据间的联系。
